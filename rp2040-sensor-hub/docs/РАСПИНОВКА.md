# Распиновка подключения датчиков к Waveshare RP2040-Zero

Документ ниже относится именно к плате **Waveshare RP2040-Zero**, а не к Raspberry Pi Pico и не к абстрактному RP2040-модулю.

## Что важно именно для RP2040-Zero

- На плате нет обычного user LED на `GPIO25`
- На плате есть встроенный **WS2812 RGB LED** на `GPIO16`
- Питание датчиков и подтяжка OneWire должны идти от **3V3**, а не от `VSYS/5V`
- Обмен с Orange Pi идёт через встроенный USB Type-C разъём платы, без `TX/RX` проводов

## Распиновка датчиков

### SHT21/HTU21 (I2C - Влажность и температура)
**Сенсор:** 4-контактный модуль или DIP-8

| Контакт SHT21 | RP2040-Zero | Назначение |
|---|---|---|
| VDD (1) | `3V3` (`P1-21`) | Питание 3.3V |
| GND (4) | `GND` (`P1-22` или `P3-5`) | Земля |
| SDA (2) | `GP4` (`P1-5`) | I2C Data |
| SCL (3) | `GP5` (`P1-6`) | I2C Clock |

**I2C адрес:** 0x40

**Важно:** SHT21/HTU21 - это 3.3V-устройство. На `VSYS/5V` его подключать не нужно.

### DS18B20 (OneWire - Температура)
**Сенсор:** TO-92 или водонепроницаемый модуль

| Контакт DS18B20 | RP2040-Zero | Назначение |
|---|---|---|
| GND (1) | `GND` (`P1-22` или `P3-5`) | Земля |
| DQ (2) | `GP6` (`P1-7`) | OneWire Data |
| VDD (3) | `3V3` (`P1-21`) | Питание 3.3V |

**Pull-up резистор:** `4.7кОм` между `GP6` и `3V3` (обязателен)

### Встроенный светодиод платы

| Узел платы | GPIO |
|---|---|
| Onboard WS2812 RGB LED | `GP16` |

`GP16` лучше считать занятым платой и не использовать под датчики.

### Подключение к Orange Pi по штатному USB
**Кабель:** обычный USB Type-C <-> USB Orange Pi

| Что подключаем | Куда |
|---|---|
| USB Type-C разъём RP2040-Zero | Любой штатный USB-порт Orange Pi |

**Важно:** для обмена данными используются встроенный USB-контроллер RP2040 и USB CDC (`/dev/ttyACM*`).  
GPIO `GP0/GP1` для связи с Orange Pi не нужны и остаются свободными.

## Электрическая схема

```
                    3V3
                      │
                      ├─────[4.7k]─────┐
                      │                 │
                      ▼                 ▼
    ┌─────────────────┴─────────────────┤
    │                                    │
   VDD                            GP6 (OneWire)
    │                                    │
 SHT21                          DS18B20  │
    │                                    │
   GND                                  GND
    │
    ├─ I2C SDA → GP4
    ├─ I2C SCL → GP5
    └─ GND

Orange Pi:
  USB порт Orange Pi ↔ USB Type-C разъём RP2040-Zero
```

## Алгоритм инициализации

1. **Питание:** RP2040-Zero получает питание по USB, а датчики питаются от `3V3`
2. **I2C инициализация:** `GP4` (`SDA`) и `GP5` (`SCL`) на `I2C0`
3. **OneWire инициализация:** `GP6` с внешней подтяжкой `4.7кОм` к `3V3`
4. **USB serial инициализация:** RP2040 появляется на Orange Pi как `/dev/ttyACM*`

## Проверка связи

```bash
# На RP2040 будет отправляться каждую секунду:
{"temp_sht21": 22.5, "humid_sht21": 45.2, "temp_ds18b20": 23.1, "timestamp": 1234567890}
```

## Безопасность и надёжность

- **Не подавайте 5V на GPIO:** подтяжка OneWire и питание SHT21 должны быть только от `3V3`
- **Развязывающие конденсаторы:** 100нФ рядом с каждым датчиком
- **Кабель USB:** обычный качественный USB-C кабель для питания и CDC-связи
